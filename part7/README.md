1)install router
-----------------
>npm install react-router-dom

>rm -rf .git

3) install webpack 
-------------------
>npm install --save-dev webpack webpack-cli>

>npm install style-loader css-loader --save-dev
>npm install --save-dev webpack-dev-server
>npm install axios
>npm start

4) start production
>cd build
>npx static-server (goto http://localhost:9080 )
